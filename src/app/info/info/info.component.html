Descripción del proyecto

Es un proyecto basado en Angular como herramienta para generar el
frontend, que a su vez consume la API de Pokemon, de la que
obtiene su funcionalidad. Es una aplicación que permite a los usuarios buscar y
acceder a los detalles de cada Pokemon.

Feedback del proyecto

Este proyecto ha sido una experiencia increíble y gratificante para mí. Me permitió integrar y utilizar
mis conocimientos en Angular y la API de Pokémon para crear una aplicación interactiva y visualmente
atractiva. La capacidad de obtener datos en tiempo real y presentar información detallada sobre cada
Pokémon ha sido un desafío emocionante y educativo. Estoy especialmente orgullosa de cómo he logrado
estilizar las cartas de Pokémon para que reflejen los tipos de cada uno y añadir elementos dinámicos como
las imágenes GIF animadas. Este proyecto no solo muestra mis habilidades técnicas, sino también mi
dedicación y pasión por la programación y el diseño de interfaces de usuario. Espero que disfruten
explorando la aplicación tanto como yo disfruté desarrollándola.
